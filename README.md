# prime-number-finder
Java library that find the next closest prime number from any number given, using odd composite numbers.

### Test with:
```java
$ ./run.sh 76876

1) Finding next prime from [76876.0]
2) The next prime number from [76876.0] is [76883.0] (0.0) ms
```

### Run Tests with:
```java
./testFile.sh
./testNextPrime.sh
```
